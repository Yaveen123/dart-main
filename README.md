Hey there!


This is the repo that I've used to dump all of my material that I've covered in the Learn The Dart Programming Language tutorial by John Elder. Completing this tutorial is a project as part of by bigger goals of being able to code applications in Flutter without the reliance of AI tools, so when I deploy Flutter applications, I can fix bugs and cybersecurity concerns without opening new holes.
